# 课程作业提交管理小工具

一个基于 React 的课程作业管理系统，帮助大学生更好地规划和跟踪学习任务。

## 📖 项目简介

本项目是一个轻量级的前端单页应用，用于管理课程作业的提交和进度跟踪。所有数据存储在浏览器本地，无需后端服务。

### ✨ 主要功能

#### 基础功能
- ✅ 添加课程作业（课程名、标题、截止日期、完成状态）
- ✅ 编辑已有作业信息
- ✅ 删除作业（带确认提示）
- ✅ 标记作业完成状态
- ✅ 按课程筛选作业
- ✅ 按"已完成/未完成"状态筛选
- ✅ 数据本地持久化（localStorage）

#### 扩展功能
- ⭐ **按截止日期排序** - 支持升序和降序排列
- ⭐ **逾期任务高亮显示** - 超过截止日期且未完成的任务用醒目红色标记

## 🛠️ 技术栈

- **框架**: React 18 (函数组件 + Hooks)
- **状态管理**: useState + useEffect
- **样式**: CSS Modules
- **数据存储**: localStorage
- **构建工具**: Create React App

## 🎯 功能演示

### 1. 添加作业
- 点击"添加新作业"按钮
- 填写课程名、作业标题、截止日期
- 点击"添加作业"保存

### 2. 编辑作业
- 点击作业卡片上的"编辑"按钮
- 修改相关信息
- 点击"保存修改"

### 3. 删除作业
- 点击"删除"按钮
- 确认删除操作

### 4. 标记完成
- 点击"标记完成"切换状态
- 已完成作业显示绿色标签

### 5. 筛选作业
- **按课程筛选**: 选择特定课程查看
- **按状态筛选**: 查看"全部"、"已完成"、"未完成"
- **组合筛选**: 同时使用课程和状态筛选

### 6. 排序作业
- 点击"升序 ↑"或"降序 ↓"按钮
- 按截止日期排序，了解最紧急任务

### 7. 逾期提示
- 逾期未完成的作业自动显示红色边框和背景
- 一眼看出需要紧急处理的任务

## 💾 数据存储

所有作业数据存储在浏览器 localStorage 中：
- **存储键名**: `assignment_manager_data`
- **数据格式**: JSON数组
- **持久化**: 页面刷新或关闭后数据仍然保留

## 📋 数据模型

```javascript
Assignment {
  id: string,          // 唯一标识
  courseName: string,  // 课程名
  title: string,       // 作业标题
  deadline: string,    // 截止日期 (ISO格式)
  completed: boolean,  // 完成状态
  createdAt: string,   // 创建时间
  updatedAt: string    // 更新时间
}
```

## 🔧 开发说明

### 组件职责
- **App.js**: 主应用，整合所有子组件和业务逻辑
- **AssignmentCard**: 展示单个作业信息和操作按钮
- **AssignmentForm**: 添加/编辑作业的模态框表单
- **FilterBar**: 筛选和排序功能
- **ConfirmDialog**: 删除确认对话框

### 自定义Hook
- **useAssignments**: 封装所有作业的增删改查操作

### 工具函数
- **storage.js**: localStorage读写操作
- **dateUtils.js**: 日期处理、逾期判断、排序比较

## 🎨 设计特点

- **简洁界面**: 清晰的信息展示，无冗余元素
- **直观操作**: 所见即所得，学习成本低
- **响应式**: 适配不同屏幕尺寸
- **视觉反馈**: 逾期任务醒目提示，完成状态清晰区分
