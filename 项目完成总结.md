## 🎨 技术亮点

### 1. 架构设计
- **职责分离**: 数据层(utils) + 业务层(hooks) + 展示层(components)
- **单一职责**: 每个组件职责清晰，易于维护
- **可复用**: 工具函数独立，可在其他项目复用

### 2. 状态管理
- **自定义Hook**: useAssignments封装所有作业操作
- **响应式更新**: React状态更新自动触发重渲染
- **持久化存储**: localStorage同步保存

### 3. 用户体验
- **逾期提示**: 醒目的红色边框和背景
- **相对日期**: "还剩3天"、"已逾期2天"
- **空状态友好**: "暂无作业，点击添加"
- **操作确认**: 删除前二次确认

### 4. 代码质量
- **命名清晰**: 文件、函数、变量名语义化
- **注释适度**: 关键逻辑有说明，不过度注释
- **错误处理**: localStorage操作有try-catch
- **性能优化**: 使用useMemo缓存筛选结果

## 🚀 如何使用

### 快速启动

```bash
cd project
npm install
npm start
```

访问 http://localhost:3000

### 功能演示流程

1. **添加第一个作业**
   - 点击"+ 添加新作业"
   - 填写：课程名="数据结构"，标题="实验报告"，截止日期="2026-06-20"
   - 点击"添加作业"

2. **体验筛选功能**
   - 添加几个不同课程的作业
   - 使用课程筛选查看特定课程
   - 使用状态筛选查看"未完成"

3. **体验排序功能**
   - 添加不同截止日期的作业
   - 点击"升序 ↑"查看最近截止的在前
   - 点击"降序 ↓"查看最远截止的在前

4. **体验逾期提示**
   - 添加一个截止日期为过去的作业
   - 观察红色边框和背景的高亮效果
   - 标记为已完成，观察高亮消失
